В программе доступны следующие роли:

Администратор
Менеджер
Авторизованный клиент
Неавторизованный клиент
У администратора есть следующие возможности: просмотр (с фильтрацией, сортировкой, поиском)/добавление/редактирование/удаление товаров, просмотр /добавление/редактирование/удаление заказов.

У менеджера есть следующие возможности: просмотр товаров (с фильтрацией, сортировкой, поиском), просмотр заказов.

У авторизованного клиент есть следующие возможности: просмотр товаров (без фильтрации, сортировки, поиска).

У неавторизованного клиента есть следующие возможности: просмотр товаров (без фильтрации, сортировки, поиска)

Как запустить программу:

скачайте файл с базой данных с расширением .sql
выполните sql-скрипт для создания базы данных
запустите .exe файл
введите нужные логин и пароль
